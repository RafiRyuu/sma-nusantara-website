# SMA Nusantara Cendekia — Official Website

Website profil sekolah premium yang dibangun dengan pendekatan modern, responsif, dan interaktif. Proyek ini dirancang sebagai platform informasi terpadu untuk calon siswa, orang tua, dan masyarakat.

## ✨ Fitur Utama

- **Premium Design**: Skema warna Navy-Gold-White yang elegan dengan tipografi tajam.
- **PPDB Online Ready**: Halaman pendaftaran lengkap dengan alur, rincian biaya, dan FAQ.
- **Gallery Masonry**: Galeri interaktif untuk menampilkan karya dan prestasi siswa.
- **Responsive & Mobile First**: Tampilan optimal di desktop, tablet, dan smartphone.
- **Micro-interactions**: Animasi halus pada tombol, menu navigasi, dan scroll reveal effect.
- **SEO Optimized**: Metadata lengkap dan Open Graph tags untuk kemudahan pencarian.
- **WhatsApp Integration**: Tombol kontak terapung yang tersedia di seluruh halaman.

## 🚀 Teknologi yang Digunakan

- **Frontend**: HTML5, Vanilla CSS, Modern JavaScript (ES6+)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Fonts**: Google Fonts (Poppins & Inter)

## 🛠️ Cara Menjalankan Proyek

1. **Clone repositori** (setelah Anda meng-upload-nya):
   ```bash
   git clone https://github.com/USERNAME/sma-nusantara.git
   ```

2. **Instal dependensi**:
   ```bash
   npm install
   ```

3. **Jalankan mode pengembangan**:
   ```bash
   npm run dev
   ```

4. **Build untuk produksi**:
   ```bash
   npm run build
   ```

## 📂 Struktur Folder

- `src/css/` : Berisi file styling (variables, base, components, per-halaman).
- `src/js/`  : Logika interaktif dan animasi.
- `public/`  : Aset gambar dan media.
- `docs/`    : Dokumentasi roadmap dan perencanaan proyek.
